# AllForAlzheimer's Website

This repository contains the source code for the official AllForAlzheimer's website.  
AllForAlzheimer's is a nonprofit organization dedicated to supporting individuals affected by Alzheimer's disease through care packages, awareness initiatives, and community engagement.

---

## Overview

The website is designed to be both informative and engaging, with the following primary sections:

- **Home** – Introduction to the organization and mission statement.
- **About** – Background, goals, and story behind AllForAlzheimer's.
- **Blog/Updates** – News, updates, and posts about our projects and initiatives.
- **Volunteer** – Sign-up area for volunteering opportunities.
- **Donations** – One-time and recurring donations, integrated with PayPal.
- **Photo Gallery** – Highlights from events, care package distributions, and community efforts.
- **Contact** – Contact form and chatbot for easy communication.

---

## Features

- Modern, responsive design suitable for desktop and mobile.
- Clear donation options with PayPal integration.
- Volunteer sign-up and contact form functionality.
- Smooth animations and transitions for a professional experience.
- Data dashboard for tracking outreach and impact.
- Accessible and empathetic design, reflecting the values of the organization.

---

## Tech Stack

- **Next.js** – Framework for building the site.
- **React** – Frontend components.
- **Tailwind CSS** – Styling and responsive design.
- **Vercel** – Hosting and deployment.
- **MDX/Markdown** – For blog posts and updates.

---

## Project Structure

```yaml
src:
  components: Reusable UI components
  pages:
    index: Homepage
    about: About the organization
    blog: Updates and posts
    volunteer: Volunteer sign-up
    donations: Donation options
    gallery: Photo gallery
    contact: Contact form and chatbot
  styles: CSS & Tailwind configuration
  utils: Helpers and configuration
public: Static assets (logos, images, icons)
README.md: Project documentation
```
